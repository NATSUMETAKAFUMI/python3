# 行列の関数

## モジュール名
 import numpy.linalg import det,inv,solve

---

# 関数

| 関数  | 機能 |
| --- | --- |
| diag() | 対角要素を成分とする1次元配列を返す |
| trace() | 対角要素の和（トレース：跡）を返す |
| det() | 行列式を返す |
| eig() | 正方行列の固有値と固有ベクトルを返す |
| inv() | 正方行列の逆行列を返す |
| pinv() | 行列のMoore-Penrose疑似逆行列を返す |
| qr() | 行列をQR分解した結果を返す |
| svd() | 行列を特異値分解した結果を返す |
| solve() | 連立1次方程式を返す |
| lstsq() | 連立1次方程式の最小自乗解を求める |
